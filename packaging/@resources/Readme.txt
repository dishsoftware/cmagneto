Thank you for installing EnowContacts!

This library provides a clean and extensible C++ API for managing contact data (name, phone, email, etc.)
and is suitable for desktop applications or services that need contact management capabilities.

📦 Installation Path:
By default, the library is installed to:
  C:\Program Files\EnowSW\Contacts
